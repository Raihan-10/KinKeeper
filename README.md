# Project Kinkeeper
## Description:
```
A personal relationship management platform  where people can easily check how when he got connected with someone else, it can be his friends,collegue or anyone else. Also when he messaged or text or video called to his friends can be checked from the timeline page and the ratio also can be seen from the stats page. This is basically one kind of tracker.
```
### Technologies
1. Html
2. Css (tailwind)
3. React
4. NextJs

### Key features
1. User can track when he got connected with someone
2. Most used features like video call, text or message
3. The time of messaging, calling or texting someone
